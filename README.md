# 프로젝트 소개
**Run & Learn** 은 RPG Maker MZ와 AI(OpenAI API)를 결합하여 만든 교육용 RPG 프로젝트입니다.  
플레이어는 게임 속 NPC와 대화하면서 AI가 실시간으로 생성한 문제를 풀고, 게임을 즐기며 자연스럽게 학습할 수 있습니다.
